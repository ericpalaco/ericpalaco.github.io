<h1>List of HTML commands to remember:</h1>
<br>
<br>
<h4>head:</h4> creates the head of the HTML document
<br>
<br>
<h4>body:</h4> creates the body of the HTML document
<br>
<br>
<h4>b:</h4> to em<b>bold</b>en closer text bold text
<br>
<br>
<h4>em:</h4> <em> to emphasize text in the HTML document</em>
<br>
<br>
<h4>strong</h4> to add some <strong>bold text</strong> to the HTML document
<br>
<br>
<h4>h1, h2, h3, h...:</h4> used to define the level of headings, (1 = largest) (3 = smallest)
<br>
<br>
<br>
<h3>Lists</h3>
<br>
<br>
<h4>ul, ol:</h4> used to create and ordered and unordered list to an HTML document
<br>
<br>
<br>
<h3>Hypertext</h3>
<br>
<br>
<h4> a:</h4> used to create a link to another document, ex: (open a) href"your link"> <samp> sample text </samp>(close a)
<br>
<br>
<br>
<h3> Images </h3>
<br>
<br>
<br>
<h4> img: </h4> loads an image to the HTML document  ex: ( open img) src="your image" alt="your title"(close)
<br>
<br>
<br>
<h3>Breaks and others</h3><br>
<br>
<br>
<h4>p:</h4> creates a paragraph
<br>
<br>
<h4>br:</h4> forces a break in the HTMl document
